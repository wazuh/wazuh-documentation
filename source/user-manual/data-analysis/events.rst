Events
======

Events are raw logs and security records collected by Wazuh agents and other log sources. After collection, the events are forwarded to the Wazuh manager, where the normalization engine decodes, normalizes, and enriches them into structured JSON documents for analysis and threat detection.

The process includes the following stages:

#. Raw log: The original log generated by the source.

#. Input event: The raw log after it is collected by the Wazuh agent and forwarded to the Wazuh manager, with Wazuh agent and cluster metadata added.

#. Normalized event: The final schema-conformant JSON document produced by the Wazuh normalization engine after decoding, normalization, and enrichment.
