verify-agent-conf
=================

The **verify-agent-conf** program verifies the Wazuh :doc:`agent.conf </user-manual/reference/centralized-configuration>` configuration.

Usage
-----

.. code-block:: console

      verify-agent-conf [-f <agent.conf file>]
