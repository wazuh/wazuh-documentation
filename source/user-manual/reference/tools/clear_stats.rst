clear_stats
===========

The clear_stats program clears the events stats.

+--------+------------------------------+
| **-h** | Display the help message.    |
+--------+------------------------------+
| **-a** | Clear all the average stats. |
+--------+------------------------------+
| **-d** | Clear the daily averages.    |
+--------+------------------------------+
| **-w** | Clear the weekly averages.   |
+--------+------------------------------+
