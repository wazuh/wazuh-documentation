Wazuh dashboard
===============

The Wazuh dashboard is a central component for analyzing, visualizing, and managing security operations. It provides a unified interface for monitoring and interpreting security data collected from multiple sources. Users can create dynamic visualizations, customize dashboards, investigate security events, and manage Wazuh platform configurations through an intuitive interface.

The Wazuh dashboard supports monitoring of system health, security events, and alerts. It also enables configuration management, compliance monitoring, threat detection, and incident investigation, helping security teams strengthen their organization’s cybersecurity posture.

Browser compatibility
---------------------

The Wazuh dashboard supports the following web browsers:

- Chrome 95 or later
- Firefox 93 or later
- Safari 14 or later

Other Chromium-based browsers might also work. Internet Explorer 11 is not supported.
